## 六、DMIT 网络线路详解

### CN2 GIA（AS4809）

CN2 GIA 是中国电信旗下的顶级商业专线，具有：

- 低延迟、高稳定性
- 晚高峰不掉速
- 价格昂贵，需要高端套餐

### CMIN2（AS58807）

中国移动国际精品线路，特点：

- 移动用户优化极佳
- 价格比 CN2 GIA 更低
- 部分线路高峰期有一定拥塞

### GIA 线路（gia network）

GIA = Global Internet Access，表示流量在骨干网层面直连，避免绕路，保障中国访问速度。

## 九、DMIT VPS 使用常见问题

### 流量超了怎么办？

DMIT 流量超额后有以下选项：
1. **升级套餐**：可以在控制面板升级到更大流量套餐
2. **购买附加流量包**：部分套餐支持单独购买流量
3. **等待月份重置**：流量按自然月重置

### 如何更换 IP？

- IP 被墙后，DMIT 政策允许 **每 15 天免费换一次 IP**
- 之后每次更换收取 $5 费用
- 可在后台控制面板（login 后操作）申请

### 如何开启 IPv6？

DMIT 默认分配 IPv6 /64 前缀，在系统网络配置中启用即可；部分套餐支持 IPv6 中国优化（CMIN2 IPv6）。

### SSH 密钥登录教程

DMIT 支持 SSH RSA 公钥登录，步骤：

bash
# 本地生成 SSH 密钥
ssh-keygen -t rsa -b 4096

# 将公钥上传到服务器
ssh-copy-id -i ~/.ssh/id_rsa.pub root@your-dmit-ip

# 登录
ssh -i ~/.ssh/id_rsa root@your-dmit-ip


Mac 用户同样使用上述命令；Windows 用户可用 PuTTY 并转换 PPK 格式。

### 退款政策

DMIT 提供有限退款保障：新购 VPS 如有问题，可在规定时间内联系客服申请退款，具体以官网政策为准。
